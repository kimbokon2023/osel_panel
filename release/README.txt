OSEL Panel Generator v1.0
========================

패널 제작 도면 자동 생성 프로그램

사용 방법:
---------
1. "excel_files" 폴더에 제작산출결과 엑셀 파일(.xlsx)을 넣어주세요.
2. "OSEL_Panel_Generator.exe" 파일을 더블클릭하여 실행하세요.
3. 프로그램이 자동으로 엑셀 파일을 읽고 DXF 도면을 생성합니다.
4. 생성된 DXF 파일은 "done" 폴더에 저장됩니다.

필요한 파일 구조:
---------------
- OSEL_Panel_Generator.exe (메인 프로그램)
- dimstyle/ (DXF 스타일 폴더)
  └── style.dxf
- excel_files/ (엑셀 파일 입력 폴더)
  └── *.xlsx (제작산출결과 파일)
- done/ (DXF 파일 출력 폴더) - 자동 생성됨

주의사항:
--------
- 엑셀 파일은 "제작산출결과" 시트가 있어야 합니다.
- 엑셀 파일이 열려있으면 오류가 발생할 수 있습니다.
- 프로그램 실행 시 엑셀 파일을 닫아주세요.

문제 해결:
---------
- 프로그램이 실행되지 않으면 관리자 권한으로 실행해보세요.
- 엑셀 파일 오류가 발생하면 파일 형식을 확인해주세요.
- DXF 파일이 생성되지 않으면 엑셀 데이터를 확인해주세요.
